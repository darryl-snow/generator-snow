## Instructions

To install Bower and Component dependencies:

```
gulp install
```

During development (compile, watch for changes, run a server):

```
gulp
```

For deployment (will package and optimise everything):

```
gulp deploy
```

To add a new page:

```
yo snow:page <pagename>
```
